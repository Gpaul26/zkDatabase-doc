# Sign-Up

The `signUp` method is used to register a new user with zkDatabase. During sign-up, the user must provide an email address, this email will be used for authentication purposes on the GUI management tool.

### Definition

```ts
await zkdb.auth.signUp(email: string): Promise<TUserSignUpResponse>;
```

### Parameters

- **`email`** (String): The user's email address.

### Returns

- A promise that resolves when the user is successfully registered.

```ts
type TUserSignUpResponse = {
  userName: string;
  email: string;
  publicKey: string;
  activated: boolean;
  userData: Record<string, any> | null;
  createdAt: Date;
  updatedAt: Date;
};
```

### Example

<DatabaseExample
  code={`\nif (!(await zkdb.auth.isUserExist("chiro-user"))) {
    await zkdb.auth.signUp("chiro@example.com");
}`}
/>

In this example, the `signUp` method registers a new user with the username `chiro-user` and email `chiro@example.com`. The signer must be setup beforehand to authenticate the sign-up process see [Setup your credentials](../getting-stated.mdx#setup-your-credentials).
